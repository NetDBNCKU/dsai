## Getting Started


### Prerequisite
- Webpack: 3.x
- webpack-dev-server: 2.x
- ESLint: [airbnb](https://github.com/airbnb/javascript/tree/master/packages/eslint-config-airbnb)


### Installing

```
$  npm install // or yarn install
```

### Run Locally (Dev)
```
$ npm run dev
```

### Generate Production Static Files
```
$ npm run build
```
